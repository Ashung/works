Automate Sketch

  • 2015/09 - 2020
  • JavaScript, CocoaScript

Automate 是一个拥有很多实用功能的 Sketch 插件。它内置很多增强功能，和很多针对团队协作的组件、样式、库的功能。

截至目前已有近 240 多个功能，几乎囊括了 Sketch 各个方面的高效操作。许多国内外知名公司的设计师都是它的深度用户，也成为很多设计师的推荐插件。

Android Res Export

  • 2016/07 - 2020
  • JavaScript, CocoaScript, HTML, CSS

我在很早便接触 Sketch 和 Sketch 插件开发，发布过多个开源插件，例如 Swatches 和 HSL Color Picker，这两个项目因 Sketch 的升级引入相同功能而停止维护。

Android Res Export 是为了解决公司的设计团队工具转到 Sketch 之后，在输出 Android 资源上遇到的问题，经过几次迭代更新，目前已经支持几乎全部的 Android 资源格式，包括多种尺寸的 PNG / WebP 位图、多种尺寸的点九图、响应式应用图标、Vector Drawable 矢量资源、点九图预览、形状和色彩代码等等。

Android Res Export 是目前国内很多手机厂商设计团队和 Android App 设计师的必备插件。

Import Colors

  • 2019/02 - 2020
  • JavaScript

Import Colors 可以帮助设计师直接导入 Photoshop，Illustrator 等软件的色板文件，也能通过纯文本格式共享色板。

通过将色板导入为库的功能，轻松解决 Sketch 没有类似 Photoshop 色彩面板的痛点。

Android Resources Export

  • 2019/07 - 2020
  • TypeScript, HTML, CSS

Android Resources Export 是 Figma 插件内测期间，首批上线的插件之一，累计至今已有 8.4k 安装用户。

插件用于导出 PNG，点九资源，响应式图标等资源，解决了 Figma 无法将多种尺寸资源导出到 Android 特定的文件结构，以及处理点九资源多种尺寸的问题。

MixFonts

  • 2019/08 - 2020
  • JavaScript, HTML, SCSS

MixFonts 用于中西文混排，为文本图层中的中文和西文指定不同的字体。可以保存多种预设组合。

Data from Local

  • 2019/09 - 2020
  • TypeScript, JavaScript, HTML, CSS

Data from Local 是一款使用本地的文本和图片的数据填充插件。

Frame Resizer

  • 2020/02 - 2020
  • TypeScript, HTML, SCSS

Frame 尺寸修改插件，Frame Resizer 解决了 Figma 默认没有按方向调整尺寸的功能，以及无法在不修改内部图层约束的情况快速调整至特定尺寸而不影响内部图层。

其中尺寸预设功能，可以快速批量修改 Frame 尺寸。

Android Vector Drawable

  • 2020/02 - 2020
  • TypeScript, JavaScript, HTML, CSS

Android Vector Drawable 用于实时查看选中图层的 Vector Drawable 代码，以及复制和导出代码功能。
