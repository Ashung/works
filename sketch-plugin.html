Android Res Export

  • 2016/07 - ...
  • Sketch
  • JavaScript, CocoaScript, HTML, CSS
  • MIT
  • 项目主页

插件支持全部的 Android 资源格式，包括多种尺寸的 PNG / WebP 位图、Nine-patch 位图、响应式应用图标、Vector Drawable 矢量资源、Nine-patch 效果预览、形状和色彩代码。是国内手机厂商设计团队和 Android App 设计师的必备插件。

  • PNG / WebP
  • Nine-patch
  • Vector Drawable
  • Adaptive Icon
  • Shape XML
  • Color XML
